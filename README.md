# AI Detective Memory Simulation

This Python script simulates the core concept of 'unforgettable AI memory' like Cognee, as described in the DetectiveAI article. It demonstrates how disparate pieces of crime investigation data (persons, events, objects, statements) can be stored, linked, and queried to reveal connections that might otherwise be missed. The example ingests various facts and then allows querying for all information related to a specific entity, mimicking an AI's ability to maintain a persistent, interconnected knowledge base.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
